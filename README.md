1. 简介
douban_list_spider.py是一个简单的爬虫，可以根据关键字抓取豆瓣电影、豆瓣读书或豆瓣音乐的条目信息.

2. Python
本人的Python版本为：2.6.6
还需要安装必要的Python插件：
# easy_install requests
# easy_install BeautifulSoup4

3. 执行抓取动作
首先对douban_list_spider.py中的变量object、tag_list和page_num进行配置。
然后执行命令即可：
# python douban_list_spider.py
